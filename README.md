# 👶 云养娃 (Cloud Baby)

> 一款从出生到高考的养成模拟游戏

## 🎮 游戏介绍

体验养育孩子的完整历程！从出生到18岁高考，48回合的策略养成，每一个决定都将影响孩子的未来。

### ✨ 游戏特色

- **📅 48回合养成** - 从出生到高考的人生旅程
- **🎓 15种不同结局** - 清华北大、985名校、艺术路线、网红路线...
- **📊 六大核心属性** - 智商、情商、才艺、魅力、学业、自律
- **🎲 35%事件触发** - 丰富的随机事件增加可玩性
- **🍼 五个成长阶段** - 婴儿期→幼儿园→小学→初中→高中
- **💰 经济系统** - 每回合工资收入，平衡收支
- **📱 完美适配手机** - 随时随地云养娃

## 📝 游戏机制

### 六大属性
| 属性 | 作用 |
|------|------|
| 🧠 智商 | 影响学习效率和高考成绩 |
| 💬 情商 | 影响社交和事件选择 |
| 🎨 才艺 | 艺术路线的关键属性 |
| ✨ 魅力 | 网红路线的重要属性 |
| 📚 学业 | 直接决定高考分数 |
| 📏 自律 | 影响稳定性和效率 |

### 面板属性
| 属性 | 作用 |
|------|------|
| 😊 情绪 | 低情绪容易抑郁、叛逆 |
| ❤️ 健康 | 健康<20会休学养病 |
| 💪 体力 | 体力不足无法行动 |
| 💰 金钱 | 用于报班、医疗、消费 |

### 成长阶段
- **👶 婴儿期** (0-1岁) - 打好基础
- **🧒 幼儿园** (1-6岁) - 性格养成
- **👦 小学** (6-12岁) - 学业起步
- **🎒 初中** (12-15岁) - 青春期挑战
- **📚 高中** (15-18岁) - 高考冲刺

## 🎯 结局类型

### 学术路线
- 🏆 清华北大 (720分+)
- 🥇 985名校 (680分+)
- 🥈 211大学 (620分+)

- ⚡ 体育路线 - 健康>75 && 体力>75
- 📱 网红路线 - 魅力>75 && 情商>60

- 💰 富二代精英 - 金钱>20万+属性要求
- 🏡 中产舒适 - 金钱>8万
- 😷 休学养病 - 健康<20
- 💔 心理问题 - 情绪<15
